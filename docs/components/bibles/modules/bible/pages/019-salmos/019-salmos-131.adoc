[[v19-131-0]]
=== Salmos, 131
// rc69 ?, RC69 - Almeida Revista e Corrigida 1969

[[v19-131-1]]

^<<v19-131-1,1>>^ SENHOR, o meu coração não se elevou nem os meus olhos se levantaram: não me exercito em grandes assuntos, nem em cousas muito elevadas para mim.

[[v19-131-2]]

^<<v19-131-2,2>>^ De certo fiz calar e sossegar a minha alma: qual criança desmamada, para com sua mãe, tal é a minha alma para comigo.

[[v19-131-3]]

^<<v19-131-3,3>>^ Espere Israel no Senhor, desde agora e para sempre.
